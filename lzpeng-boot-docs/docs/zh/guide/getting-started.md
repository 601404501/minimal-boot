# 快速上手
## 后端
```bash
# 克隆项目
git clone https://github.com/lzpeng723/lzpeng-boot-server.git
# 修改src/main/resources/application-mysql.yml中数据库配置信息
# 进入项目目录
cd lzpeng-boot-server
# 启动项目
mvn springboot:run

```
## 前端
```bash
# 克隆项目
git clone https://github.com/lzpeng723/lzpeng-boot-client.git
# 进入项目目录
cd lzpeng-boot-client
# 安装依赖 建议使用 cnpm 安装
npm i -S
# 启动项目
npm run dev
```
